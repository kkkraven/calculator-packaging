1. Установи Node.js (если не установлен)
2. Установи Wrangler: npm install -g wrangler
3. В терминале запусти: wrangler dev
4. Открой новое окно, перейди в папку с проектом, запусти: npm install, затем npm run dev
5. Открой сайт по адресу http://localhost:5173
6. Чтобы деплоить в облако — используй wrangler deploy

Пояснение для разработчиков:
- Всё, что не связано с обменом данными с Gemini, не менять!
- Базу знаний и правила расчёта подтягивать как в оригинальном проекте
- Если нужны дополнительные данные — использовать файлы constants.ts и все существующие структуры
